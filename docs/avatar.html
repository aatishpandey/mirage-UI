Avatar
Avatars are basically user's representation which can be used to display user image.
Profile Avatars
Profile Avatars can be created just by using avatar, avatar-s class in your html document for img tag
Profile Avatars with border
Profile Avatars can be created just by using avatar,avatar-s and border class based on color like bd-red class in your html document for img tag
Profile Avatars with different size
Profile Avatars can be created for different size by using avatar,avatar-s and size class based on sizes like avatar-l,avatar-m.
