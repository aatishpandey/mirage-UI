Buttons
Buttons are one of the most important and used components for any website.
Primary Buttons
Primary Buttons can be created using btn-primary class along with the btn class.
Secondary Buttons
Secondary Buttons can be created using btn-secondary class along with the btn class.
Link Buttons
Link Buttons can be created using btn-link class along with the btn class.
Icon Buttons
Icon Buttons can be created using btn-icon class along with the btn class.
Floating Buttons
Icon Buttons can be created using btn-floating-action class along with the btn-container-float class for parent .
